Easily change the order of the meta boxes on the posts screen.

== Description ==
Easily customize the order of meta boxes on posts screen.

*If you want to force users, set $force_override  as true

That's all, Check source code for more explanation.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from admin's Plugins page.
== Frequently Asked Questions ==

*   I changed meta boxes order but no effected users blog?
	-  Don't forget to change $posts_widgets_order_hash after the update
*   Is it compatible with multisite?
	- Definitely!
*   How can prevent to users change order?
	- Set $force_override variable as true
